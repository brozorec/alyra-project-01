# alyra-project-01

### Clone and install packages
```
git clone https://github.com/brozorec/alyra-project-01.git
cd alyra-project-01
npm install
cd client
npm install
```

### Deploy contracts (open ganache)
```
// in new terminal
truffle deploy
```

### Run front-end
```
// in new terminal
cd client
yarn start
```
